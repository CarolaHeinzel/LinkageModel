# LinkageModel

This repository contains the code for the paper [A Statistical Test for Comparing the Linkage and Admixture Model Based
on Central Limit Theorems](https://arxiv.org/abs/2509.12734). It contains code to evaluate and apply the statistical test in this paper. <br>


Additionally, it contains example data, which have the correct format for the statistical test. These are the 1000 Geneomes Data .<br>

## Application of the code

To apply the code, please download the files and run them in e.g. spyder. <br>
